# ICP Scoring Criteria

## Scoring Model

Each newly funded company is scored on a 0-10 scale:

### Funding Amount (weight: 25%)
- $1M-$5M: 6/10
- $5M-$20M: 8/10
- $20M-$50M: 10/10
- $50M+: 9/10 (often too large for our solution)

### Company Size (weight: 25%)
- 10-50 employees: 7/10
- 50-200 employees: 10/10
- 200-500 employees: 8/10
- 500+: 5/10

### Industry Match (weight: 30%)
- Exact match (SaaS, Fintech): 10/10
- Adjacent (E-commerce, MarTech): 7/10
- Other tech: 4/10
- Non-tech: 1/10

### Tech Stack Indicators (weight: 20%)
- Uses modern stack (cloud, APIs): 8/10
- Has engineering blog/GitHub: +1
- Job postings for automation roles: +1

## Qualification Threshold

Score >= 7.0 → Auto-create lead + start outreach
Score 5.0-6.9 → Create lead, manual review
Score < 5.0 → Log only, no action
